# Installation Guide

This guide explains how to set up and run the Password Manager project locally.

---

## Requirements

- Python **3.11+**
- `pip`
- Virtual environment support (recommended)

---

## Setup Steps

### 1. Clone the repository

```bash
git clone git@github.com:stefan0413/Python_Password_Manager_2FA.git
                        OR
git clone https://github.com/stefan0413/Python_Password_Manager_2FA.git

cd Python_Password_Manager_2FA
```

### 2. Install the needed dependencies

```bash
pip install .
```

### 3. Run the project

```bash
python -m password_manager.main
